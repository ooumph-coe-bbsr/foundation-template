2. **Verify before you use it** — never submit raw AI output as your own. Write down how you checked: ran the code, checked the textbook, compared a second tool, tested with real data.
3. **Link long threads** — log the key prompt here and put the full conversation in [`prompt-log.md`](prompt-log.md), referenced by its Prompt Log ID.
4. **Missing log = module rejected** — a module with no `AI_USAGE` entry cannot be graded.
5. **Oral defence** — you must be able to explain any AI-assisted work live. If you can't, it is disqualified.
